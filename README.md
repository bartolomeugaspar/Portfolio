## 🚀 Sobre o Projeto

Este portfólio foi desenvolvido com foco em design moderno, experiência do usuário e funcionalidades dinâmicas. O site apresenta uma interface limpa e profissional, com animações suaves e um sistema de gerenciamento de projetos integrado.

### ✨ Destaques

- Design responsivo e moderno com efeitos glassmorphism
- Sistema dinâmico para adicionar novos projetos
- Modais interativos para exibição detalhada dos projetos
- Integração com WhatsApp para contato direto
- Armazenamento local dos projetos adicionados
- Animações e transições suaves

## 🎯 Funcionalidades

### 🏠 Página Principal
- Apresentação pessoal com foto e informações básicas
- Download direto do CV em PDF
- Menu de navegação fixo e responsivo

### 🛠️ Seção de Habilidades
- Exibição visual das tecnologias dominadas
- Ícones das principais linguagens e ferramentas
- Layout organizado e responsivo

### 📂 Gerenciamento de Projetos
- **Visualização**: Cards organizados com imagens e descrições
- **Detalhes**: Modais com informações completas dos projetos
- **Adição Dinâmica**: Sistema para adicionar novos projetos via modal
- **Upload de Imagens**: Possibilidade de fazer upload de imagens locais
- **Persistência**: Projetos salvos no localStorage do navegador

### 📞 Contato
- Formulário integrado com WhatsApp
- Links para redes sociais
- Envio direto de mensagens

## 💻 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica

## 🎮 Como Usar

### Acesso Online
**🌐 [https://bartolomeugaspar.github.io/Portfolio/](https://bartolomeugaspar.github.io/Portfolio/)**

### Visualização Local

1. **Clone o repositório**
   ```bash
   git clone https://github.com/bartolomeugaspar/Portfolio.git
   cd Portfolio
   ```

2. **Abra o projeto**
   - Abra o arquivo `index.html` em qualquer navegador moderno
   - Ou use um servidor local como Live Server (VS Code)

### Adicionando Novos Projetos

1. **Clique no botão "Adicionar Projeto"** (canto superior direito)
2. **Preencha o formulário** com:
   - Nome do projeto
   - Descrição detalhada
   - Imagem (upload de arquivo local)
   - Link do projeto
3. **Clique em "Adicionar"** - O projeto aparecerá imediatamente
4. **Persistência**: Os projetos ficam salvos mesmo após recarregar a página

### Personalizando o Portfólio

1. **Informações Pessoais**: Edite o arquivo `index.html`
   - Altere nome, foto, descrição
   - Atualize links das redes sociais
   - Substitua o arquivo CV

2. **Projetos Existentes**: Modifique os cards na seção de projetos

3. **Estilos**: Customize as cores e estilos no arquivo `styles.css`
